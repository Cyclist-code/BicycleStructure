# Bicycle Structure
An Android application for novice cyclists about the structure of a bicycle.
## Technologies and Tools
- Programming language C#;
- Platform [Xamarin.Forms](https://github.com/xamarin/Xamarin.Forms);
- Visual Studio 2022.
